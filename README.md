# Mostrador Eleitoral

forked from https://github.com/DyeffersonAz/mostrador-eleicao

Consulta e mostra informações sobre a apuração das eleições municipais de 2020 no site do TSE sobre as cidades do Vale do Paraiba.

## Instalação:

instalar o Nodejs

e rodar

```
npm i
```

## Utilização:

```
node index.js
```

acessar:  

localhost:3000
